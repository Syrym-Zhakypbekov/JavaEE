   BATCH FILE PRACTICE EXERCISES
═══════════════════════════════════════════════════

This folder contains simple practice batch files
to help you learn how .bat files work.

FILES:
------

1. 01-hello.bat
   - Basic echo commands
   - Shows date, time, computer name
   - Run: Double-click or type "01-hello.bat"

2. 02-variables.bat
   - How to use variables
   - How to get user input
   - Run: Double-click

3. 03-menu.bat
   - Interactive menu system
   - Using goto and labels
   - Run: Double-click and try all options

HOW TO RUN:
-----------
Method 1: Double-click the .bat file
Method 2: Open Command Prompt, navigate here, type filename

EXERCISE:
---------
Try modifying these files!
1. Right-click → Edit (opens in Notepad)
2. Change some text
3. Save
4. Run again to see changes
